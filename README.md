# HTML5 WebGL FPS Demo

纯 HTML5 + 原生 JavaScript + WebGL2 的 3D 第一人称射击演示（无第三方库）。

## 运行

浏览器对 `Pointer Lock` + `WebGL` 通常要求通过 HTTP 打开（不要直接双击 `index.html`）。

在项目目录运行：

```bash
python3 -m http.server 5173
```

然后打开：

```
http://localhost:5173/
```

## 操作

- 点击页面进入游戏（Pointer Lock）
- W/A/S/D：移动
- Space：跳跃
- Shift：下蹲
- Alt：冲刺
- 鼠标左键：射击
- R：换弹
- 1 / 2：切换武器
- B：切换开火模式（AUTO/SEMI）
- G：投掷烟雾墙（仅爆破模式，带冷却）
- U：购买护甲（仅爆破冻结期）
- I：购买步枪（仅爆破冻结期）
- O：购买烟雾（仅爆破冻结期）
- Esc：退出指针锁定（暂停）

## 说明

这不是《CS:GO》的完整复刻（完整地图/枪械/网络/动画/音效/经济系统等规模巨大且涉及版权）。
当前版本提供一个可玩的单机 FPS 基础骨架，后续可以继续扩展：更多武器、AI、关卡、贴图、音效、多人联机等。

## 当前地图与出生规则

- 当前版本已固定为爆破玩法，不再提供 Endless / Match 分支模式。
- 地图已扩展为更大场地，并采用多链路布局（上路 / 中路 / 下路 + 多个转点）。
- 玩家与 Bot 使用分区出生：CT 与 T 各自从本阵营出生区随机点位出生与重生。
- Bot 数量设置当前按“每队人数”计，例如 `5v5` 表示每队 5 人。
- 默认对战模式为爆破（Bomb），启动 AI 后会直接进入回合制流程（冻结期 -> 交战 -> 回合结算）。
- 爆破模式为 A/B 双点：T 方可在进入点位时选择对应点位下包，CT 方需在已下包点位回防拆包。
- 中路新增高台与双侧转点，A/B 区域增加门口与回旋掩体，减少单一直线交火。
- 爆破模式新增真回合制：开局冻结时间、回合时间、回合结算与基础经济系统（击杀/胜负奖励）。
