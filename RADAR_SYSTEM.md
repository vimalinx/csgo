# CSGO 雷达/小地图系统

## 概述

实现了完整的雷达/小地图系统，提供实时的战场态势感知能力。

## 功能特性

### 1. 左上角小地图

**尺寸**：
- 小模式：200x200px
- 大模式：300x300px
- 隐藏模式：不显示

**显示内容**：
- ✅ 地图轮廓（简化版障碍物）
- ✅ 玩家位置（蓝点，带朝向箭头）
- ✅ 队友位置（蓝点，带朝向箭头）
- ✅ 敌人位置（红点，仅当可见时显示）
- ✅ 半透明背景（不遮挡视野）

### 2. 雷达功能

#### 敌人可见性检测
- ✅ 敌人仅在以下情况显示：
  - 在玩家视野内（无障碍物阻挡）
  - 被队友看到（队友视野内）

**实现原理**：
- 使用射线投射（Raycasting）检测视线
- 沿玩家-敌人连线采样多个点
- 检查每个采样点是否与障碍物碰撞
- 同时检查所有存活队友的视线

#### 包点标记
- ✅ A包点和B包点位置显示
- ✅ 金色圆圈标记包点范围
- ✅ 包点字母标识（A/B）

#### 炸弹标记
- ✅ 炸弹放置后显示位置
- ✅ 红色圆点 + 十字标记
- ✅ 高对比度，易于识别

### 3. 交互控制

**Tab 键切换大小**：
```
循环模式：200px → 300px → 隐藏 → 200px ...
```

**状态显示**：
- 右下角显示当前雷达尺寸
- 显示操作提示 `[Tab 切换]`

## 技术实现

### 文件结构

```
csgo/
├── radar.js              # 雷达系统核心类
├── main.js               # 集成雷达系统
├── style.css             # 雷达样式
└── RADAR_SYSTEM.md       # 本文档
```

### 核心类：Radar

**位置**：`radar.js`

**主要方法**：

#### `constructor(host, gameState)`
- 初始化雷达画布
- 设置事件监听器
- 绑定游戏状态

#### `worldToMap(x, z, bounds)`
- 将世界坐标转换为雷达坐标
- 支持动态地图边界

#### `lineOfSightBlocked(start, end, boxes)`
- 检查两点间视线是否被阻挡
- 使用射线投射算法
- 每 0.5 单位采样一次

#### `isEnemyVisible(enemyPos, playerPos, bots, boxes)`
- 检查敌人是否可见
- 检查玩家视野
- 检查所有队友视野

#### `render()`
- 主渲染函数
- 绘制地图、玩家、敌人、包点、炸弹
- 每帧调用

### 视觉设计

**颜色方案**：
```javascript
背景：rgba(6, 10, 16, 0.75)     // 深色半透明
边框：rgba(255, 255, 255, 0.25) // 浅色边框
玩家：#4f9cff                   // 蓝色
队友：TEAM_VISUALS[team].hex    // 根据队伍颜色
敌人：TEAM_VISUALS[team].hex    // 根据队伍颜色
包点：rgba(255, 215, 0, 0.6)    // 金色
炸弹：rgba(255, 60, 60, 0.95)   // 红色
障碍物：rgba(170, 188, 214, 0.2) // 浅蓝灰色
```

**响应式设计**：
- 根据屏幕大小调整边距
- 小屏幕优化布局

## 性能优化

### 1. 视线检测优化
- ✅ 采样步长优化（0.5 单位）
- ✅ 提前终止（发现障碍物立即返回）
- ✅ 只检测存活角色

### 2. 渲染优化
- ✅ 使用 `desynchronized: true` 提升性能
- ✅ Canvas 2D 硬件加速
- ✅ 只渲染可见元素

### 3. 事件优化
- ✅ 使用 `keydown` 而非 `keyup`（更快响应）
- ✅ `e.repeat` 检查防止重复触发

## 使用指南

### 玩家操作

1. **查看雷达**：默认左上角显示
2. **切换大小**：按 `Tab` 键
3. **识别目标**：
   - 蓝色箭头 = 自己
   - 蓝点带箭头 = 队友（箭头指向朝向）
   - 红点 = 可见敌人
   - 金色圆圈 + 字母 = 包点
   - 红色十字 = 炸弹

### 开发者集成

**初始化**：
```javascript
import Radar from './radar.js'

const radar = new Radar(hudElement, gameState)
```

**渲染循环**：
```javascript
function render() {
  // ... 其他渲染代码
  radar.render()
}
```

**切换大小**：
```javascript
const newSize = radar.toggleSize()
// 返回: 200, 300, 或 0 (隐藏)
```
