### 🔴 高危问题（P0）

#### 1. 事件监听器泄漏
- **位置：** main.js:4764-4929
- **问题：** 30+ addEventListener，几乎无 removeEventListener
- **影响：** 内存泄漏、性能下降、逻辑错误
- **修复建议：**
  ```javascript
  // 在 returnToLobby() 中添加：
  globalEventManager.clear('window');
  globalEventManager.clear('document');
  globalEventManager.clear('game');
  ```

#### 2. 超长函数
- `makeBot()` - **1262 行**（line 889）
- `drawWorld()` - **547 行**（line 6350）
- `updateWeapon()` - **325 行**（line 5042）
- **修复建议：** 拆分为多个小函数

#### 3. 除零风险
- **位置：** v3norm 函数（line 149）
- **问题：** L=0 时导致除零错误
- **修复建议：**
  ```javascript
  function v3norm(a) {
    const L = Math.sqrt(a.x * a.x + a.y * a.y + a.z * a.z);
    if (L < 1e-8) return v3(0, 0, 0);
    return { x: a.x / L, y: a.y / L, z: a.z / L };
  }
  ```

### 🟡 中危问题（P1）

1. **对象池无容量限制** - 可能无限增长
2. **A* 算法性能瓶颈** - 使用数组而非优先队列
3. **WebSocket 错误处理不完整** - 缺少重连机制
4. **数组越界风险** - 多处未检查长度
5. **浮点数精度问题** - 累积误差
6. **输入验证不足** - 参数未检查
7. **重复函数定义** - switchEquip, currentEquip

### 📊 性能瓶颈

1. **Bot 更新频率过高** - 虽有节流，仍可优化
2. **碰撞检测复杂度** - 射线检测 O(n)
3. **渲染调用未优化** - 无实例化渲染
4. **内存分配频繁** - 临时对象过多
