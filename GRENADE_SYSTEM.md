# CSGO 投掷物系统

## 概述

本系统实现了 CSGO 风格的投掷物系统，包含三种投掷物：闪光弹、烟雾弹和燃烧弹。

## 功能特性

### 1. 闪光弹（Flashbang）

- **键位**：4 键切换，左键投掷
- **价格**：$200
- **最大库存**：2 发
- **冷却时间**：14 秒
- **效果**：
  - 爆炸半径：14 米
  - 致盲效果：白屏 + 逐渐恢复
  - 恢复时间：2.5 秒
  - 对敌人造成 2.8 秒的射击冷却

### 2. 烟雾弹（Smoke）

- **键位**：5 键切换，左键投掷
- **价格**：$300
- **最大库存**：2 发
- **冷却时间**：16 秒
- **效果**：
  - 烟雾半径：4 米
  - 烟雾高度：2.5 米
  - 持续时间：15 秒
  - 粒子数量：200 个
  - 漂浮动画效果
  - 阻挡视线（射线检测）

### 3. 燃烧弹（Molotov/Incendiary）

- **键位**：6 键切换，左键投掷
- **价格**：$400
- **最大库存**：1 发
- **冷却时间**：18 秒
- **效果**：
  - 火焰半径：3 米
  - 火焰高度：1.5 米
  - 持续时间：7 秒
  - 每秒伤害：8 HP
  - 粒子数量：150 个
  - 火焰颜色渐变（红->橙->黄）

## 技术实现

### 投掷轨迹

- 使用物理引擎模拟抛物线轨迹
- 重力加速度：12.0 m/s²
- 空气阻力系数：0.995
- 最大反弹次数：3 次
- 最大飞行时间：3 秒

### 粒子系统

- 基于 Canvas 2D 实现
- 3D -> 2D 投影渲染
- 粒子生命周期管理
- 实时更新和渲染

### 碰撞检测

- 与现有碰撞系统兼容
- AABB 包围盒检测
- 地面碰撞检测
- 墙壁反弹处理

## 使用方法

### 购买

在冻结期（freeze time）按 B 打开购买菜单：

- Q 键：购买闪光弹 ($200)
- W 键：购买烟雾弹 ($300)
- R 键：购买燃烧弹 ($400)

### 投掷

1. 按数字键切换投掷物类型：
   - 4 键：闪光弹
   - 5 键：烟雾弹
   - 6 键：燃烧弹
2. 左键投掷
3. 投掷方向跟随准星

### 战术运用

- **闪光弹**：突破时致盲敌人，创造进攻机会
- **烟雾弹**：遮挡狙击线，掩护移动
- **燃烧弹**：封锁区域，阻止敌人推进

## 性能优化

1. **粒子池化**：预分配粒子数组，避免频繁内存分配
2. **视锥剔除**：只渲染屏幕内的粒子
3. **生命周期管理**：自动清理过期效果
4. **碰撞优化**：使用 AABB 快速检测
