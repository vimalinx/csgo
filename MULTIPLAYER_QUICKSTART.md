# 🎮 CSGO 多人在线 - 快速开始指南

## 🚀 立即开始

### 方式 1: 直接在游戏中体验

1. **启动游戏**
   ```bash
   # 在浏览器中打开
   firefox index.html
   # 或
   google-chrome index.html
   ```

2. **进入多人模式**
   - 点击 "联机模式" 按钮
   - 等待连接服务器（自动）

3. **注册账号**
   - 输入用户名（3-20字符）
   - 点击 "进入游戏"

4. **选择房间**
   - 查看现有房间列表
   - 点击 "加入" 进入房间
   - 或创建新房间

5. **开始游戏**
   - 自动进入游戏
   - 可以看到其他玩家
   - 享受多人对战！

### 方式 2: 先测试连接

1. **打开测试页面**
   ```bash
   firefox test-multiplayer.html
   ```

2. **逐步测试**
   - 点击 "连接服务器"
   - 输入用户名并 "注册"
   - 创建或查看房间
   - 确认所有功能正常

3. **返回游戏**
   - 打开 `index.html`
   - 开始多人游戏

## ❓ 常见问题

### Q: 无法连接服务器？
A: 
1. 检查网络连接
2. 确认服务器地址正确
3. 使用 `test-multiplayer.html` 测试

### Q: 看不到其他玩家？
A: 
1. 确认都在同一个房间
2. 等待几秒钟同步
3. 检查控制台是否有错误

### Q: 延迟很高？
A: 
1. 检查网络状况
2. 尝试靠近服务器位置
3. 关闭其他网络应用

### Q: 游戏卡顿？
A: 
1. 检查浏览器性能
2. 降低图形设置
3. 关闭不必要的标签页
